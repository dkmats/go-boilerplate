# Go Boilerplate

This is my personal boilerplate for Go projects.
Feel free to use it!

## Prerequisites

This boilerplate is designed for Go projects using VS Code with Dev Containers.  
If you're new to them, check out the links below:

- Visual Studio Code: <https://code.visualstudio.com/download>
- Dev Containers: <https://code.visualstudio.com/docs/devcontainers/containers#_installation>
